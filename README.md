# ❌⭕ Tic Tac Toe - Web Edition

A simple yet elegant implementation of the classic **Tic Tac Toe** game using HTML, CSS, and JavaScript. Designed with a modern UI and smooth user experience, this game features turn tracking, win detection, and a replay option.

---

## 🎮 Features

- 🔄 **Turn-Based Play** — X and O alternate turns.
- 🧠 **Win Detection** — Automatically highlights the winning combination.
- 🤝 **Draw Support** — Detects when the game ends in a tie.
- 🎨 **Modern UI** — Styled with blurred glass effect and responsive layout.
- 🕹️ **New Game Button** — Resets the game without reloading the page.

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Play the game and enjoy!

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling with `backdrop-filter`, grid layout, transitions
- **JavaScript (ES6)** – Game logic and dynamic interactions
